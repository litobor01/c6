# c6

Браузерный калькулятор — статический HTML-сайт.

## Локальный запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```powershell
npx serve .
```

## Деплой на Vercel

### Через веб-интерфейс

1. Импортируйте репозиторий на [vercel.com](https://vercel.com).
2. Vercel автоматически определит проект как статический сайт.
3. Настройки сборки менять не нужно — build-команда не требуется.

### Через CLI

```powershell
npm i -g vercel
vercel
```

Для продакшн-деплоя:

```powershell
vercel --prod
```

Старый URL `/calculator.html` перенаправляется на главную страницу.
